## Система отображения парковок

В рамках учебного проекта был разработан веб-сайт, предоставляющий пользователям удобный способ поиска свободных парковочных мест.

Backend реализован на языке Java с использованием фреймворка Spring.
Frontend выполнен с помощью HTML, CSS и JavaScript.
Для отображения карты используется бесплатное API OpenStreetMap.
